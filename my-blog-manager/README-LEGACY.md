# Legacy Components

This directory contains the original local-manager workflow from upstream XHBlogs.

## Status

- **Deprecated in this repository**
- **Not part of the recommended maintenance path**
- kept temporarily for historical reference and rollback comparison only

## Do not use this as the default workflow

For this repository, the recommended path is now:

- edit content under `../content/`
- run the frontend from `../XHBlogs/`
- manage changes with Git

The manager-driven sync/update workflow is intentionally no longer the main route.
